Different Methods of Embedding JavaScript

1. Internal JavaScript (within <script> tags)

This method places JavaScript code directly within <script> tags in the HTML file.

        <script>
            function internalAlert() {
                alert("This is an inline JavaScript alert!");
            }
        </script>

2. External JavaScript File

This method links to an external .js file using the <script> tag with a src attribute.

        <script src="external.js"></script>

3. Inline Event Handler

This method embeds JavaScript directly in HTML event attributes.

        <button onclick="alert('This is an inline event handler alert!')">Click</button>

4. Event Listener (added via JavaScript)

This method uses JavaScript to attach event listeners to elements.

        <button id="listenerButton">Click</button>
        <script>
        document.getElementById('listenerButton').addEventListener('click', function() {
            alert('This is an event listener alert!');
        });
        </script>
